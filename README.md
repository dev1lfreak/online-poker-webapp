# online-poker-webapp

### Функциональные требования

#### Серверная часть

1. Управление подключениями
    - При подключении клиент передаёт свой никнейм, сервер регистрирует его и помещает в список ожидания или сразу за свободный стол.
    - Поддерживается одновременная работа до нескольких десятков клиентов.
2. Игровые столы
    - Сервер одновременном имеет несколько покерных столов с нефиксированным количеством мест (минимальное).
    - Игрок может автоматически попасть за свободный стол либо выбрать конкретный стол из списка.
    - Каждый стол ведёт собственную партию независимо от других.
3. Игровая логика
    - Смена дилера, блайнды. 
    - Раздача: каждому игроку выдаются две закрытые карты.
    - Общие карты открываются поэтапно: флоп (три карты), тёрн (одна), ривер (одна).
    - Игроки по очереди совершают действия: чек, колл, фолд, рейз (с возможностью ввести сумму или ползунком).
    - После завершения всех раундов торговли происходит вскрытие: сервер определяет лучшую комбинацию и объявляет победителя (или несколько при ничьей).
    - Банк перераспределяется между победителями, фишки обновляются.
4. Фишки.
	- Каждый стол - отдельный стек фишек, игра идёт до последнего оставшегося за столом.
5. Чат (опционально)
    - Игроки за одним столом могут обмениваться текстовыми сообщениями.
    - Сообщения рассылаются всем участникам стола.
6. Обработка отключений и таймаутов
    - Если игрок отключается во время игры, сервер помечает его как отсутствующего.
    - На ход отводится ограниченное время (например, 30 секунд); если игрок не успевает сделать ход, сервер автоматически выполняет чек (если это допустимо) или фолд.

#### Клиентская часть

1. Графический интерфейс
    - Окно входа: поля для ника, лобби ожидания столов.
    - Основное окно стола:
        - Изображения карт игрока (две карты) и общих карт (до пяти).
        - Панель с кнопками действий (Check, Call, Fold, Raise) и поле для ввода ставки.
        - Список игроков за столом с отображением их фишек и статуса (кто ходит, кто дилер).
        - Текущий размер банка.
        - Область чата с историей сообщений.
        - Отображение таймера.
        - Отображение рук.
